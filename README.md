### The Hard Method
```js
GET QR
$ apt update
$ apt install nodejs 
$ pkg install git
$ git clone https://github.com/vishnukayamkulam/Lalisa
$ cd Lalisa
$ chmod +x *
$ npm install github:adiwajshing/baileys
$ npm install chalk
$ node qr.js
```
